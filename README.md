# Ai-emotion-Detector

# This is an ai emotion detector which uses tensorflow to train a model using FER-2013 dataset of images 



#
Edit and run model.py to train your CNN on the organized data.

python model.py
This creates emotion_model.h5 (your trained model) in the project folder.
#


#
Start the Flask server:

python app.py
Open your browser to http://localhost:5000/
#

#
Required Packages

tensorflow or tensorflow-cpu

flask

opencv-python

numpy
#
